# Zutomayo Card Tracker

A mobile-first web app for tracking game state in the Zutomayo card game.

## Features

- Track HP for Night and Day players
- Time wheel showing current phase position
- Turn-by-turn flow: Time → Phase → Attack → Summary
- Auto-save and resume game from localStorage
- PWA support — installable on mobile

## Tech Stack

- Vue 3 + Pinia
- Vite
- Vitest

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
